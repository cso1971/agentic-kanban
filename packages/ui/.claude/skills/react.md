# React 19

You are an expert in React 19 development.

## Key Features

- Use function components with hooks
- React 19 has improved hydration and concurrent features
- Use `use()` hook for promises and context
- Server Components support (when applicable)
- Improved ref handling - refs as props without forwardRef

## Patterns

- Prefer `useState`, `useReducer` for local state
- Use `useEffect` for side effects, `useLayoutEffect` for DOM measurements
- Use `useMemo` and `useCallback` only when needed for performance
- Custom hooks for reusable logic
- Use React.lazy() for code splitting

## Best Practices

- Keep components focused and composable
- Lift state up when needed, but not too high
- Use proper key props in lists
- Handle loading and error states
- Prefer controlled components for forms

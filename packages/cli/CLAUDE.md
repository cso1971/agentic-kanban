# @agentic-kanban/cli

CLI package for running agents from the command line.

## Tech Stack

- **Runtime**: Bun
- **Language**: TypeScript (ESM)
- **Dependencies**: @agentic-kanban/core

## Commands

```bash
bun run cli          # Run the CLI
bun run typecheck    # Type check
```

## Structure

- `src/cli.ts` - Main CLI entrypoint (binary)

## Skills

Skills are defined in `.claude/skills/`.

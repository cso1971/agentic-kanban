## Overview

This library provides four fundamental arithmetic operations: addition, subtraction, multiplication, and division. All functions are fully typed with TypeScript and include comprehensive test coverage.

## Installation

```bash
bun install
```

## Usage

```typescript
import { add, subtract, multiply, divide } from '@agentic-kanban/example-calculator';

// Addition
const sum = add(5, 3);        // 8

// Subtraction
const diff = subtract(10, 4);  // 6

// Multiplication
const product = multiply(6, 7); // 42

// Division
const quotient = divide(15, 3); // 5
```

## API Reference

### `add(a: number, b: number): number`

Adds two numbers together.

**Parameters:**
- `a` - The first number
- `b` - The second number

**Returns:** The sum of `a` and `b`

**Example:**
```typescript
add(2, 3);     // 5
add(-1, -2);   // -3
add(5, 0);     // 5
```

---

### `subtract(a: number, b: number): number`

Subtracts the second number from the first.

**Parameters:**
- `a` - The number to subtract from
- `b` - The number to subtract

**Returns:** The difference of `a` minus `b`

**Example:**
```typescript
subtract(5, 3);  // 2
subtract(3, 5);  // -2
```

---

### `multiply(a: number, b: number): number`

Multiplies two numbers together.

**Parameters:**
- `a` - The first number
- `b` - The second number

**Returns:** The product of `a` and `b`

**Example:**
```typescript
multiply(3, 4);   // 12
multiply(5, 0);   // 0
multiply(-2, -3); // 6
```

---

### `divide(a: number, b: number): number`

Divides the first number by the second.

**Parameters:**
- `a` - The dividend (number to be divided)
- `b` - The divisor (number to divide by)

**Returns:** The quotient of `a` divided by `b`

**Throws:** `Error` if `b` is zero

**Example:**
```typescript
divide(10, 2);  // 5
divide(7, 2);   // 3.5

// Division by zero throws an error
divide(5, 0);   // throws Error: "Cannot divide by zero"
```

## Error Handling

Only `divide` can throw. All other operations (`add`, `subtract`, `multiply`) always return a number.

| Function | Throws | Condition |
|----------|--------|-----------|
| `add` | Never | — |
| `subtract` | Never | — |
| `multiply` | Never | — |
| `divide` | `Error("Cannot divide by zero")` | When `b === 0` |

Always guard division when the divisor may be zero:

```typescript
import { divide } from '@agentic-kanban/example-calculator';

try {
  const result = divide(numerator, denominator);
  console.log(result);
} catch (err) {
  console.error('Division failed:', (err as Error).message);
}
```

## Development

### Building

```bash
bun run build
```

This will compile the TypeScript source files to the `dist/` directory.

### Testing

```bash
bun test --pass-with-no-tests
```

The test suite includes comprehensive coverage for all arithmetic operations, including edge cases like:
- Addition with negative numbers and zero
- Subtraction with negative results
- Multiplication by zero and negative numbers
- Division with decimals and division by zero errors

### Type Checking

```bash
bun run typecheck
```

## CI/CD

This project uses GitLab CI for continuous integration. The pipeline automatically:
- Installs dependencies
- Builds the project
- Runs the test suite

The pipeline is triggered on merge requests when changes are made to the calculator example.
